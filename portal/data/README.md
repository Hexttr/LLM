# Данные каталога моделей

## model-tiers.json

Список категорий моделей для главной страницы. Редактируйте этот файл, чтобы добавлять или менять модели без правок кода.

**Формат:** массив категорий (tiers). Каждая категория:

- `id` — идентификатор: `free` | `daily` | `top`
- `title` — заголовок блока (например, «Бесплатные модели»)
- `subtitle` — краткое описание
- `textModels` — массив текстовых моделей: `{ "name": "...", "provider": "...", "description": "..." }`
- `imageModels` — массив моделей для генерации изображений в том же формате

После сохранения файла обновите страницу в браузере (API читает файл при каждом запросе).
